# Задачи

Вынести метеоданные в переменную, в отдельный файл. Данные карточек на главной странице заполнить из этой переменной.

Для описания стилей использовать методологию БЭМ.

Важно: в рамках этого задания выполняется только верстка блока карточек с заполнением данных о погоде из переменной, функционал получения актуальных данных о погоде реализовывать не требуется.

- 3я таска

* сверстать нижний блок с прогнозом «на 24 часа» и «на 5 дней».
* Минимальная ширина экрана — 1280px.
* Настроить обработчик вкладок для кнопок «на 24 часа» и «на 5 дней» в новом блоке так, чтобы при клике на кнопку срабатывало соответствующее переключение вкладок с выводом данных о погоде в карточках. Карточки вкладок заполнить данными из макета.
* Данные должны храниться в отдельных переменных для прогноза на «на 24 часа» и «на 5 дней».
* При нажатии на ссылку в футере, открывается новая вкладка с URL
  https://preax.ru
* Для описания стилей использовать методологию БЭМ.
* в рамках текущей задачи обработчик кнопки прокрутки не настраивать, только верстка элементов.
