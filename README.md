Работа с риложением

- для запуска приложения используйте **live server**
- используются svg sprite - не пугайтесь его в html
- в папке publiс скачены svg для будущего
- папка style разделена на компоненты, чтобы в будущем с удобством переездать на React или компонентный подход
- папка scripts также сделана для будущего, имеются файлы которые будут скорее всего применяться в следующих спринтах(плохо вник в задание сначала,но удалять не стал)
- Мок данные находятся в файлу db.json
- класс thumb для ползунка манипулирует через width.Сделано для того чтобы через js манипулировать положением ползунка. BG акцента реализован через относительное положение родительского компонента.Через планируется - если давление( или что то другое) , бэкграунд начинает растягиваться от центра.Класс dangers.
- Карточки генерируются через js класса **Card** , и цепляются к id weatherCards после загрузки HTML в файле script.js

# Задачи

Вынести метеоданные в переменную, в отдельный файл. Данные карточек на главной странице заполнить из этой переменной.

Для описания стилей использовать методологию БЭМ.

Важно: в рамках этого задания выполняется только верстка блока карточек с заполнением данных о погоде из переменной, функционал получения актуальных данных о погоде реализовывать не требуется.
